class LazySimArray(base_array: Optional[ndarray] = None, n_frames: int = 1)#

Bases: Sequence

A lazy array for the simulation of the data.

The simulation data is acquired frame wise. The idea is thus to register all the required operation to produce this frame.

This is very close to what is done in larray[R5d23ad48583c-1]_ libray, but evaluation will alwaysbe considered frame wise.
