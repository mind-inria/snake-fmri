AbstractHandler#

class simfmri.simulator.handlers.base.AbstractHandler[source]#

Bases: ABC

Handler Interface.

An Handler is designed to modify a Simulation data object.

Handlers can be chained using the @ operator. Once created, an handler (and its chain of other registered handler) can be applied on a simulation using the handle function

Examples

>>> A = Handler()
>>> B = Handler()
>>> C = Handler() @ A
>>> s1 = Simulation()
>>> C.handle(s1.copy()) == B.handle(A.handle(s1))
