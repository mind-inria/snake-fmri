Bases: object

Data container for a simulation.

Parameters:

  • shape (Shape2d3d) – Shape of the volume simulated

  • n_frames (int) – Number of frames acquired

  • TR (float) – Acquisition time for one volume/frame

  • n_coils (int) – Number of coils acquired, default 1

  • **extra_infos (dict) – dict

SimulationData
